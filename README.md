# goit-node-final-task

## Авторизація через Swagger

1. Відкрийте сторінку документації `http://localhost:3000/api-docs` та перейдіть до ендпоінтів авторизації.

   ![Крок 1 — форма логіну](/docs/auth/01_login.png)

2. Натисніть кнопку **Try it out** i заповніть форму логіну коректною електронною поштою та паролем. Натисніть кнопку **Execute**.

   ![Крок 2 — відправка запиту логіну](/docs/auth/02_login_try_it.png)

3. Після успішного логіну скопіюйте значення поля `token` з відповіді сервера.

   ![Крок 3 — отриманий токен](/docs/auth/03_login_token.png)

4. У правому верхньому куті Swagger натисніть кнопку **Authorize**.

   ![Крок 4 — відкриття форми Authorize](/docs/auth/04_auth.png)

5. Вставте скопійований токен у поле (без слова `Bearer`, Swagger додасть його сам) і натисніть **Authorize**, щоб аутентифікувати всі захищені запити.

   ![Крок 5 — введення токена в Authorize](/docs/auth/05_auth_token.png)
